## the application authorization 
The application will have at lest 3 role
    - Super admin
    - organization owner
    - Organization Sales
    - Guest user

## The application future base on role

A. Super admin
1. super admin can create organization 
2. a super admin can create the user with role of owner of organization, an organization only have on owner.

B. Organization Owner 
1. Organization Owner can see dashboard of conversation and ticket 
2. Organization Owner can assign a conversation to sales

C. Organization Sales
1. Organization Sales can see a dashboard of ticket and conversation assign to him/her
2. Organization Sales can have send a message on the conversation to the Guest

D. Guest
1. Guest can have an page to make a conversation to some organization he chose
2. Guest have dashboard of list of conversation he create to the organization 
3. Guest can make conversation with the sales of the organization
